# YourProjectName — вариант GR

Второй базовый проект (отличается от [work-project](https://github.com/yeeterblessedgod/work-project)):

- зелёная тема оформления
- другой набор товаров в каталоге
- порт по умолчанию **3001**
- подсказка по роли на странице каталога
- карточки товаров с полосой категории

Функционал тот же: роли guest / user / manager / admin, капча, админ-панель.
